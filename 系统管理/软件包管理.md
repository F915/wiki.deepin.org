# 软件包管理

包管理的作用是管理整个系统环境中的所有软件包。支持从远程仓库下载、安装并自动处理依赖。

deepin 是基于 Debian 的 Linux 发行版，因此使用 apt 作为包管理器，dpkg 作为包管理器基础，两者都作为系统的一部分被预装在操作系统环境中。本篇主要讲述 apt 的工作方式和使用方法。
